Instruksi Tugas GitHub Mahasiswa

Workflow yang digunakan: Fork → Clone → Edit → Commit → Push → Pull Request

1. Buka Repository Dosen

Buka Repository:

https://github.com/joysinaga04/LATIHAN-BPG.git

2. Klik Tombol “Fork”

Tombol Fork berada di pojok kanan atas GitHub.

Tujuan Fork: Menyalin repository dosen ke akun GitHub masing-masing.

3. Clone Repository Hasil Fork

Klik tombol hijau Code, lalu copy URL repository.

Contoh:

https://github.com/joysinaga04/LATIHAN-BPG.git

Buka Git Bash / CMD lalu ketik:

git clone https://github.com/joysinaga04/LATIHAN-BPG.git

4. Masuk ke Folder Repository

cd LATIHAN-APG

5. Tambahkan File HTML

Masuk ke folder: 

mahasiswa

Buat file sesuai nama masing-masing.

Contoh:

budi.html

6. Isi HTML Data Diri

Contoh isi file seperti Gambar (sesuikan dengan data diri kamu):

;

7. Simpan File

Contoh:

mahasiswa/budi.html

8. Git Add

git add .

9. Commit

git commit -m "Menambahkan halaman Budi"

10. Push ke GitHub

git push origin main

11. Buat Pull Request

Buka repository GitHub hasil fork.

Biasanya akan muncul tombol: 

Compare & Pull Request

Klik tombol tersebut.

12. Klik “Create Pull Request”

Tambahkan deskripsi singkat jika perlu.

Contoh:

Menambahkan halaman HTML data diri.

Lalu klik:

Create Pull Request

13. Selesai 🎉

Dosen akan mereview dan merge hasil pekerjaan mahasiswa.

Catatan:
Pastikan nama file menggunakan nama masing-masing.
Contoh: budi.html, sinta.html, andi.html
